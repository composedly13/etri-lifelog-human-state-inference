## 1. Introduction

> **목적**: 연구 동기, 문제 정의, 핵심 기여 제시

### 작성할 내용

- **인간 상태 이해의 복잡성**  
  수면 품질, 피로, 스트레스 등 인간 상태는 단일 센서나 단일 이벤트 감지만으로는 충분히 설명되지 않는다.  
  여러 생리·행동 신호가 복합적으로 상호작용하여 상태를 결정한다.

- **개인차와 시간적 맥락의 중요성**  
  수면, 피로, 스트레스는 개인마다 기준선이 다르고, 요일·계절·생활 리듬 등 시간적 맥락의 영향을 크게 받는다.  
  동일한 수면 시간이라도 개인과 상황에 따라 해석이 달라진다.

- **멀티모달 라이프로그의 가치**  
  스마트폰, 웨어러블, IoT 기기에서 수집되는 라이프로그 데이터는 일상에서 비개입적(non-intrusive)으로  
  인간 상태를 추론할 수 있는 핵심 데이터다.  
  ETRI 라이프로그 데이터셋은 신체 활동, 수면, 심박, 위치 등 다양한 모달리티를 포함한다.

- **본 연구의 기여**  
  본 연구는 시간적 맥락 피처(날짜/요일/lag/rolling)와 개인별 기준선 편차 피처를 결합한  
  피처 엔지니어링 전략을 통해 인간 상태 예측 성능을 향상시키는 것을 목표로 한다.  
  LightGBM 기반의 단계적 실험으로 각 피처 전략의 기여도를 실증적으로 검증한다.

## 4. Method

> **목적**: 제안 방법론 상세 설명

### 작성할 내용

- **전체 파이프라인 개요**  
  Data → Preprocessing → Feature Engineering → Model Training → Inference.  
  단계별 모듈 구조 (`src/` 참고).

- **피처 엔지니어링 전략 (핵심 기여)**

  **(a) Baseline 피처**:  
  원본 피처의 결측치 처리, 범주형 인코딩.
  
  **(b) 시간적 맥락 피처 (Baseline 2 + Proposed 1)**:  
  날짜 파생 피처 (요일, 주말 여부, 시간대, 계절).  
  Lag 피처 (전일/전주 값), Rolling 통계 (이동평균/표준편차).  
  *"일주기 리듬과 주간 패턴이 인간 상태를 예측하는 데 유의미한 정보를 제공한다."*
  
  **(c) 개인별 기준선 피처 (Proposed 2)**:  
  개인별 평균, 표준편차, deviation, z-score.  
  *"절대값이 아닌 개인 기준선 대비 편차가 상태 이상(anomaly)을 더 잘 포착한다."*

- **모델 구성**  
  LightGBM + KFold/GroupKFold cross-validation.  
  Multi-target 독립 학습 전략.

- **앙상블 (Proposed 3)**  
  LightGBM + CatBoost + XGBoost 가중 평균 앙상블.

## 5. Experiments

> **목적**: 실험 설계와 결과를 체계적으로 제시

### 작성할 내용

- **실험 설정**  
  데이터 분할 방식 (5-Fold KFold, GroupKFold).  
  평가 지표 (RMSE, MAE 등 대회 공식 metric).  
  하이퍼파라미터 고정 조건 (seed=42).

- **비교 실험 (Ablation Study)**

  | 실험 | 피처 전략 | 비고 |
  |------|----------|------|
  | Exp001 | 원본 피처 | Baseline |
  | Exp002 | + 날짜/요일/주말 피처 | +시간 컨텍스트 |
  | Exp003 | + lag/rolling 피처 | +단기 시계열 |
  | Exp004 | + 개인별 편차 피처 | +개인 기준선 |
  | Exp005 | LGB+CB+XGB 앙상블 | 최종 |

- **피처 중요도 분석**  
  각 피처 그룹이 예측에 기여하는 정도 정량화.  
  상위 N개 피처와 연구 가설과의 일치 여부.

- **GroupKFold vs KFold 비교**  
  개인 단위 분할이 validation 신뢰도에 미치는 영향.

## 6. Results and Discussion

> **목적**: 실험 결과 해석 및 연구 가설 검증

### 작성할 내용

- **전체 성능 비교 테이블**  
  Exp001 → Exp005의 OOF 점수와 LB 점수 비교.  
  각 피처 전략이 baseline 대비 개선된 정도.

- **시간적 맥락 피처의 효과**  
  날짜/요일/lag/rolling 피처가 어떤 target에서 효과적이었는가?  
  특정 요일/계절에 따른 상태 변화 패턴 해석.

- **개인별 기준선 피처의 효과**  
  개인 편차 피처가 일반화에 기여한 정도.  
  GroupKFold 결과와 일반 KFold 결과 비교 (과적합 여부).

- **Error Analysis**  
  예측 오차가 큰 경우의 특성 분석.  
  특정 target에서 성능이 낮은 이유 가설.

- **한계점**  
  현재 방법론의 제약 (시계열 분할 미적용, 단일 모달리티 등).  
  개선 가능한 방향.

## 7. Conclusion

> **목적**: 연구 요약 및 향후 연구 방향 제시

### 작성할 내용

- **연구 요약**  
  멀티모달 라이프로그 데이터를 활용한 인간 상태 추론 문제.  
  시간적 맥락 + 개인별 기준선 피처 엔지니어링 전략의 효과.  
  단계적 ablation study를 통한 각 피처의 기여도 검증.

- **핵심 발견**  
  어떤 피처 전략이 가장 효과적이었는가?  
  인간 상태 예측에서 시간적 맥락/개인차의 중요성 재확인.

- **향후 연구 방향**  
  딥러닝 기반 시계열 모델 (LSTM, Transformer) 적용 가능성.  
  실시간 스트리밍 데이터 처리.  
  실제 애플리케이션 (스마트 헬스케어, 웰니스 서비스) 적용.
